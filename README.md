Audio Play Server
=================

post mp3 URL => play


Dependencies
------------

* Mac OSX or Linux
* Sinatra
* afplay or mpg321



Install Dependencies
--------------------

    % gem install bundler
    % bundle install
    
    % sudo apt-get install mpg321 # for Ubuntu


Run Server
----------

edit "config.yaml"


run

    % ruby development.rb


for Passenger, use config.ru
